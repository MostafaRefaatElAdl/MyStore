# Installation

- Clone the project
- Open CMD inside the ```console MyStore``` Folder
- Run the following code into terminal : 
 ```console
//to install angular cli
npm install -g @angular/cli

//to install npm modules
npm install

//to open up the project
ng serve -o
```
